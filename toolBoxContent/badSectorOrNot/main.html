隨身碟/硬碟壞軌了嗎？

(for windows)



Better safe than sorry - 有備無患

你是否會擔心沒有備份資料的隨身碟或硬碟突然壞軌？ 相信我，如果是很重要且沒有備份的資料若是突然消失或是無法取得對於所有人來說肯定是心急如焚。 為了避免突然發生這種惡夢，我們還是有一些預防措施。

壞軌是硬碟上的部分區域損壞導致無法正常讀寫的區域。 壞軌又分成兩種，一種是物理性壞軌，另外一種是邏輯性壞軌。

邏輯性壞軌：主要是不正常的軟體操作導致的，是磁軌結構異常，可能可以透過格式化來修復壞軌區域

物理性壞軌：磁軌上已經有實際摩擦損壞的部分了，是無法修復的

當然囉，邏輯性壞軌也不一定可以靠格式化來修復；物理性壞軌可能因為讀寫異常或是摩擦導致壞軌。 而且物理性壞軌可怕的地方是壞軌區域會“傳染”，磁區損壞面積有極大可能會擴大，我們不可能有那麼多時間去判斷這是哪種壞軌， 因此我都會將這兩種壞軌視為最嚴重的物理性壞軌，只要我發現硬碟有壞軌，就立刻開始備份。

那要怎麼發現硬碟上有壞軌呢？今天要來跟大家介紹“Hard Disk Validator”。

這是 Hard Disk Validator 的 GitHub 倉庫， 今天要下載的東西就在這裡，請按照以下步驟下載。

  1. 點擊"Hard Disk Validator_1.1.2.zip"


  2. 解壓縮文件夾


  3. 在剛才解壓縮的檔案中點擊"net40"，接著點開應用程式


  4. 選擇要掃描的硬碟或是隨身碟，接著按下start


  5. 等待直到每一個區塊都被掃瞄過，綠色表示區塊暫時沒事，紅色表示區塊受損，必須在一切都變得太晚之前儘快備份


強烈建議常常使用的硬碟(HDD,SSD)、隨身碟、SD、micro SD... 所有的儲存裝置每兩個月檢查一次有沒有壞軌， 那些沒有很常使用的儲存裝置則是每半年左右檢查一次。

發生在我身上的鳥事

在我升上高中的時候，我為我的手機插上記憶卡，並且把所有照片都預設儲存到記憶卡上面。 天真的我以為我每年只需要備份記憶卡的照片就好了，如此一來備份將會是一個很簡單的工作。 過了一段時間，我需要找一些照片才發現文件管理系統會莫名奇妙的閃退， 不斷告知我要格式化micro SD我才能繼續使用，因此我認為我的手機可能壞掉了。 接著我將手機連上電腦去取得我要的資料，最後電腦也是一直叫我格式化micro SD， 原本讀得到的照片又馬上消失，我就重複連接電腦好幾次，試圖挽救記憶卡裡面超過三千張的照片。 突然有一個想法閃過我的腦海：有沒有可能是記憶卡壞軌了？ 因此我就用Hard Disk Validator去掃記憶卡確認問題。結果是我猜對了的確有壞軌， 但是沒想到有1/3到1/2的面積都壞軌了。

因為無法取得裡面的照片，所以我只能先將記憶卡收起來。

大概又過了半年，我又發現了這張記憶卡，拿去電腦讀讀看。 裡面大約出現了一千五百張的照片，我就馬上先把這張照片拷貝到我的電腦裡面了， 另外一半的照片因為儲存的磁軌壞掉了，最後我也只能將那張記憶卡格式化最後丟掉了。
